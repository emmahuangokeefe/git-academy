## Prep for some git fu Open a shell, and... ```bash # Install git sudo apt-get install git # Configure git git config --global user.name "my full name" git config --global user.email "your_email@example.com" git config --global color.ui auto ``` And go setup a github account http://github.com
## Git and Secure SHell (SSH) The best way to interact with github is using ssh SSH uses public/private key pairs Lets set that up quickly ```bash # Generate SSH keys ssh-keygen -t rsa -C "your_email@example.com" # You should see them in ~/.ssh ls -l ~/.ssh cat ~/.ssh/id_rsa.pub ```
Take the text from ***~/.ssh/id_rsa.pub*** and put it into github ![1 Settings, 2 SSH keys, 3 Add SSH key](img/github-ssh.png)
